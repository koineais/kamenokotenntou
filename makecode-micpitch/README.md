# micro:bit マイク音程検出拡張 (mic-pitch)

この拡張は micro:bit v2 のマイク入力を用いて簡易ピッチ検出を行い、MIDIノート番号（例: 60 = C4）または周波数（Hz）を返します。

使い方（MakeCode）:
- ブロック: `音程（MIDI）を取得` は MIDI ノート番号（整数）を返します。音が検出できないときは `0` を返します。
- ブロック: `周波数（Hz）を取得` は推定周波数を返します。検出できないときは `0` を返します。

注意:
- マイクのサンプリングはアナログピン `P0` を使って行います。micro:bit v2 の内蔵マイクが P0 に出力されている環境を想定しています。
- 精度は簡易実装のため完璧ではありません。屋外ノイズや低音域では誤差が出ます。

拡張ファイル:
- `main.ts` : 実装本体
